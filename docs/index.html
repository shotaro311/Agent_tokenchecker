リセットを先読みする。

CodexBarはセッション/週間のウィンドウ（対応時はクレジットも）をメニューバーに表示し、 「いつ安全に進められるか」を迷わず判断できるようにします。

最新版をダウンロード GitHubで見る

Homebrew: brew install --cask steipete/tap/codexbar

  • プロバイダごとに1ステータス項目（Codex / Claude / Cursor / Gemini / Antigravity / z.ai）。
  • セッション + 週間のバーとリセット時刻。
  • ブラウザCookieによるWeb上限（Codex / Claude / Cursor）。
  • 最小UI、Dockアイコンなし。

プロバイダ

Codex · Claude Code · Cursor · Gemini · Antigravity · z.ai

データソース

可能な範囲でCLI + ローカルRPC、Codex/Claude/CursorはブラウザCookieによるWebダッシュボード、 さらにz.aiのAPIキーにも対応（z.aiダッシュボード）。

プライバシー

既存のブラウザCookie（Safari/Chrome/Firefox）を再利用してダッシュボードへアクセスします。パスワードは保存しません。 Cookieが無い場合はローカルCLI出力にフォールバックします。
