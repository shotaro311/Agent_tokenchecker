# 要件定義書：UI言語切替（日本語/English）

## 目的
- アプリ内のUI/表示文言を日本語中心にし、設定で英語/日本語を切り替え可能にする。
- 初期言語は日本語とする。

## 対象スコープ
- 対象:
  - メニューバーのメニュー項目・アクション名
  - 設定画面（タブ名/見出し/説明/ボタン/注意文）
  - 使用量/ステータス/通知/エラーメッセージなど、アプリ内で生成する文言
  - 日付/相対時刻/通貨などの表示フォーマット
- 非対象:
  - TODO: 外部から取得する生データ（メール/プラン名/モデル名）の翻訳可否を決める
  - TODO: CLI出力/ログ/ドキュメントの翻訳対象可否を決める

## 機能要件
- 設定画面に「言語」選択（日本語/English）を追加する。
- 言語選択は永続化され、再起動後も保持される。
- 初回起動/未設定時は日本語が選ばれている。
- 言語変更はアプリのUI/表示に反映される。

## 完了条件
- 設定で日本語/Englishを切り替えられる。
- 初期表示が日本語になっている。
- UI/表示文言が選択言語に応じて切り替わる。
- `swift test` / `pnpm check` が成功する。
